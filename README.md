# super-list

This is an app for managing all your todos in one place. It will track your progress when using the checkboxes inside the lists. It will save all your changes to the cloud the moment you change anything. You can change the order of the todos if you hold and drag a todo up or down. You can also delete a todo by swiping right on it. You can rename your lists if you go inside one. Renaming individual todos is not supported, you'll need to delete the todo and create a new one.
